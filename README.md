# Bank Management System (EMU8086)

**Author:** Hashir Ali  
**Course:** CSE2006 – Microprocessor and Interfacing

---

## Overview

A simple **Bank Management System** in **x86 Assembly** using EMU8086.  
Supports creating accounts, depositing & withdrawing money, viewing details, and updating account information.

---

## Features

- Create New Account
- View Account Details (PIN protected)
- Withdraw / Deposit Money (fixed amounts)
- Reset Account
- Update Account Details
- Exit Program

---

## Usage

1. Open `BankSystem.asm` in EMU8086.  
2. Assemble and run.  
3. Navigate using the numeric menu.  
4. Enter PIN for secure operations.

---

## Notes

- All accounts are saved in `users.txt`.  
- Designed for educational purposes and learning Assembly programming.
